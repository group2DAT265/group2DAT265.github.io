static void org.jabref.gui.ClipBoardManager.addX11Support ( TextInputControl  input)
inlinestatic

Add X11 clipboard support to a text input control. It is necessary to call this method in every input where you want to use it:

ClipBoardManager.addX11Support(TextInputControl input);

.

Parameters
inputthe TextInputControl (e.g., TextField, TextArea, and children) where adding this functionality.
