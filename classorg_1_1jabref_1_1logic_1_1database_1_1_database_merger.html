Merges all entries and strings of the other database into the target database. Any duplicates are ignored. In case a string has a different content, it is added with a new unique name. The unique name is generated by suffix "_i", where i runs from 1 onwards.

Merges all entries, strings, and metaData of the other database context into the target database context. Any duplicates are ignored. In case a string has a different content, it is added with a new unique name. The unique name is generated by suffix "_i", where i runs from 1 onwards.
