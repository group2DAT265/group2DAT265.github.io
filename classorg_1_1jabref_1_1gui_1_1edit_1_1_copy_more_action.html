void org.jabref.gui.edit.CopyMoreAction.copyKeyAndLink ( )

This method will copy each selected entry's citation key as a hyperlink to its url to the clipboard. In case an entry doesn't have a citation key it will not be copied. In case an entry doesn't have an url this will only copy the citation key.
