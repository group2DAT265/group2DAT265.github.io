static List<BibEntry> org.jabref.model.database.BibDatabases.purgeEmptyEntries ( Collection< BibEntry >  entries)
inlinestatic

Receives a Collection of BibEntry instances, iterates through them, and removes all entries that have no fields set. This is useful for rooting out an unsucessful import (wrong format) that returns a number of empty entries.
