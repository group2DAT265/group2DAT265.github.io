Detailed Description

This class represents all supported MSBib entry types.

Book, BookSection, JournalArticle, ArticleInAPeriodical, ConferenceProceedings, Report, InternetSite, DocumentFromInternetSite, ElectronicSource, Art, SoundRecording, Performance, Film, Interview, Patent, Case, Misc

See BIBFORM.XML, shared-bibliography.xsd (ECMA standard)
