Detailed Description

Inserts the current date (the time a database is being exported).

If a fieldText is given, it must be a valid DateTimeFormatter pattern. If none is given, the format pattern will be yyyy-MM-dd hh:mm:ss z. This follows ISO-8601. Reason: https://xkcd.com/1179/.
