Creates an object that can undo/redo an edit event.

Parameters
groupsRootThe global groups root.
editTypeThe type of editing (ADD_NODE, REMOVE_NODE_KEEP_CHILDREN, REMOVE_NODE_AND_CHILDREN)
editedNodeThe edited node (which was added or will be removed). The node must be a descendant of node groupsRoot! This means that, in case of adding, you first have to add it to the tree, then call this constructor. When removing, you first have to call this constructor, then remove the node.
