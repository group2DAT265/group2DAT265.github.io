void org.jabref.gui.fieldeditors.EditorTextArea.addToContextMenu ( final Supplier< List< MenuItem >>  items)
inline

Adds the given list of menu items to the context menu. The usage of Supplier prevents that the menus need to be instantiated at this point. They are populated when the user needs them which prevents many unnecessary allocations when the main table is just scrolled with the entry editor open.

Implements org.jabref.gui.fieldeditors.ContextMenuAddable.
