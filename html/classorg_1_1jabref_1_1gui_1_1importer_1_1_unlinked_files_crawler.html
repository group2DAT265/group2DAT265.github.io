)
inlineprivate

Searches recursively all files in the specified directory.

All files matched by the given UnlinkedPDFFileFilter are taken into the resulting tree.

The result will be a tree structure of nodes of the type CheckBoxTreeItem.

The user objects that are attached to the nodes is the FileNodeWrapper, which wraps the File-Object.

For ensuring the capability to cancel the work of this recursive method, the first position in the integer array 'state' must be set to 1, to keep the recursion running. When the states value changes, the method will resolve its recursion and return what it has saved so far.
