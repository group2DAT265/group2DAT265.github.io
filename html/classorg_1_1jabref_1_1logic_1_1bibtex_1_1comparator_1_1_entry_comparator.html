Detailed Description

This implementation of Comparator takes care of most of the details of sorting BibTeX entries in JabRef. It is structured as a node in a linked list of comparators, where each node can contain a link to a new comparator that decides the ordering (by recursion) if this one can't find a difference. The next node, if any, is given at construction time, and an arbitrary number of nodes can be included. If the entries are equal by this comparator, and there is no next entry, the entries' unique IDs will decide the ordering.
