Detailed Description

This class provides methods to create default JavaFX dialogs which will also work on top of Swing windows. The created dialogs are instances of the FXDialog class. The available dialogs in this class are useful for displaying small information graphic dialogs rather than complex windows. For more complex dialogs it is advised to rather create a new sub class of FXDialog.
