◆ getJabRefFormat()

String org.jabref.model.entry.Month.getJabRefFormat ( )
inline

Returns the month in JabRef format. The format is the short 3-digit name surrounded by a '#'. Example: #jan#, #feb#, etc.

See Issue 263 for a discussion on that thing. This seems to be an invalid format in terms of plain BiBTeX, but a valid format in the case of JabRef. The documentation is available at the Strings help of JabRef.

Returns
Month in JabRef format
