CSL Locales Repository

github.com/citation-style-language/locales is the official repository for Citation Style Language (CSL) locale files and is maintained by CSL project members. For more information, check out CitationStyles.org and the repository wiki.

Licensing

All the locale files in this repository are released under the Creative Commons Attribution-ShareAlike 3.0 Unported license. For attribution, any software using CSL locale files from this repository must include a clear mention of the CSL project and a link to CitationStyles.org. When distributing these locale files, the listings of translators in the locale metadata must be kept as is.
