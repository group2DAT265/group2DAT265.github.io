Detailed Description

FileFilter implementation, that allows only files which are not linked in any of the BibEntrys of the specified BibDatabase.

This FileFilter sits on top of another FileFilter -implementation, which it first consults. Only if this major filefilter has accepted a file, this implementation will verify on that file.
