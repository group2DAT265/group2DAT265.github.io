Detailed Description

Backups the given bib database file from BibDatabaseContext on every BibDatabaseContextChangedEvent. An intelligent ExecutorService with a BlockingQueue prevents a high load while making backups and rejects all redundant backup tasks. This class does not manage the .bak file which is created when opening a database.
