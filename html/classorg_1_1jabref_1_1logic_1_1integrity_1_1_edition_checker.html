Checks, if field contains only an integer or a literal (biblatex mode) Checks, if the first letter is capitalized (BibTeX mode) biblatex package documentation: The edition of a printed publication. This must be an integer, not an ordinal. It is also possible to give the edition as a literal string, for example "Third, revised and expanded edition". Official BibTeX specification: The edition of a book-for example, "Second". This should be an ordinal, and should have the first letter capitalized.
