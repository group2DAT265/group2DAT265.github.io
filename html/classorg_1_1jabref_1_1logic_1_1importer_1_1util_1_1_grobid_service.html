Detailed Description

Implements an API to a GROBID server, as described at https://grobid.readthedocs.io/en/latest/Grobid-service/#grobid-web-services

Note: Currently a custom GROBID server is used... https://github.com/NikodemKch/grobid

The methods are structured to match the GROBID server api. Each method corresponds to a GROBID service request. Only the ones already used are already implemented.
