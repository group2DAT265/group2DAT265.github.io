Detailed Description

This class transforms ordinal numbers into LaTeX superscripts.

Member Function Documentation

◆ format()

String org.jabref.logic.formatter.bibtexfields.OrdinalsToSuperscriptFormatter.format ( String  value)
inline

Converts ordinal numbers to superscripts, e.g. 1st, 2nd or 3rd. Will replace ordinal numbers even if they are semantically wrong, e.g. 21rd

1st Conf. Cloud Computing -> 1\textsuperscript{st} Conf. Cloud Computing
