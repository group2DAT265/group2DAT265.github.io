inlineprivate

Queries the API.

If only

searchQuery

is given, then the API will return results for each article that matches the query. If only

ids

is given, then the API will return results for each article in the list. If both

searchQuery

and

ids

are given, then the API will return each article in

ids

that matches

searchQuery

. This allows the API to act as a results filter.

Parameters
searchQuerythe search query used to find articles; details
idsa list of arXiv identifiers
startthe index of the first returned result (zero-based)
maxResultsthe number of maximal results (has to be smaller than 2000)
Returns
the response from the API as a XML document (Atom 1.0)
Exceptions
FetcherExceptionif there was a problem while building the URL or the API was not accessible
