Detailed Description

Saves the given BibDatabaseContext on every BibDatabaseContextChangedEvent by posting a new AutosaveEvent. An intelligent ScheduledThreadPoolExecutor prevents a high load while saving and rejects all redundant save tasks. The scheduled action is stored and canceled if a newer save action is proposed.
