◆ acceptableDrop()

boolean org.jabref.gui.groups.GroupNodeViewModel.acceptableDrop ( Dragboard  dragboard)
inline

Decides if the content stored in the given Dragboard can be droped on the given target row. Currently, the following sources are allowed:

  • another group (will be added as subgroup on drop)
  • entries if the group implements GroupEntryChanger (will be assigned to group on drop)
