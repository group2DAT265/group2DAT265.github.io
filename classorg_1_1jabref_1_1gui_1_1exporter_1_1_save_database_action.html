Detailed Description

Action for the "Save" and "Save as" operations called from BasePanel. This class is also used for save operations when closing a database or quitting the applications.

The save operation is loaded off of the GUI thread using BackgroundTask. Callers can query whether the operation was canceled, or whether it was successful.
