inlinestatic

Try to write the given BibTexEntry in the XMP-stream of the given PDF-file.

Throws an IOException if the file cannot be read or written, so the user can remove a lock or cancel the operation.

The method will overwrite existing BibTeX-XMP-data, but keep other existing metadata.

This is a convenience method for writeXMP(File, Collection).

Parameters
fileThe path to write to.
entryThe entry to write.
databasemaybenull An optional database which the given bibtex entries belong to, which will be used to resolve strings. If the database is null the strings will not be resolved.
Exceptions
TransformerExceptionIf the entry was malformed or unsupported.
IOExceptionIf the file could not be written to or could not be found.
