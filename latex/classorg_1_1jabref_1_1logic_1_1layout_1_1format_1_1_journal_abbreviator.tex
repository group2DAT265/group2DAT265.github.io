\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1layout_1_1format_1_1_journal_abbreviator}{Journal\+Abbreviator}} formats the given text in an abbreviated form according to the journal abbreviation lists.

The given input text is abbreviated according to the journal abbreviation lists. If no abbreviation for input is found (e.\+g. not in list or already abbreviated), the input will be returned unmodified.
