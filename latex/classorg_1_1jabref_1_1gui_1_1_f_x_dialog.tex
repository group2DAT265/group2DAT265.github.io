\doxysubsection{Detailed Description}
This class provides a super class for all dialogs implemented in Java\+FX. 

To create a custom Java\+FX dialog one should create an instance of this class and set a dialog pane through the inherited \mbox{\hyperlink{}{Dialog\#set\+Dialog\+Pane(\+Dialog\+Pane)}} method. The dialog can be shown via \mbox{\hyperlink{org_8citationstyles_8locales_8txt_a353c815edde1bbfb63a8b0b1dfb855de}{Dialog\#show()}} or \mbox{\hyperlink{interfaceorg_1_1jabref_1_1gui_1_1_dialog_acc8f8506bdfc7a77cb22154496231eb0}{Dialog\#show\+And\+Wait()}}. 

The layout of the pane should be defined in an external fxml file and loaded it via the \mbox{\hyperlink{}{F\+X\+M\+L\+Loader}}.
