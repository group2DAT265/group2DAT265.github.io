\doxysubsection{Detailed Description}
This view model can be used both for \char`\"{}add exporter\char`\"{} and \char`\"{}modify exporter\char`\"{} functionalities. It takes an optional exporter which is empty for \char`\"{}add exporter,\char`\"{} and takes the selected exporter for \char`\"{}modify exporter.\char`\"{} It returns an optional exporter which empty if an invalid or no exporter is created, and otherwise contains the exporter to be added or that is modified.
