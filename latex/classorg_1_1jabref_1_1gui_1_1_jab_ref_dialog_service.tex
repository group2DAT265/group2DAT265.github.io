\doxysubsection{Detailed Description}
This class provides methods to create default Java\+FX dialogs which will also work on top of Swing windows. The created dialogs are instances of the \mbox{\hyperlink{classorg_1_1jabref_1_1gui_1_1_f_x_dialog}{F\+X\+Dialog}} class. The available dialogs in this class are useful for displaying small information graphic dialogs rather than complex windows. For more complex dialogs it is advised to rather create a new sub class of \mbox{\hyperlink{classorg_1_1jabref_1_1gui_1_1_f_x_dialog}{F\+X\+Dialog}}.
