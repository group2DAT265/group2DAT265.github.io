\doxysubsection{Detailed Description}
This formatter takes two arguments and examines the field text. If the field text represents multiple individuals, that is it contains the string \char`\"{}and\char`\"{} then the field text is replaced with the first argument, otherwise it is replaced with the second. For example\+:

\textbackslash{}format\mbox{[}\mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1layout_1_1format_1_1_if_plural}{If\+Plural}}(Eds.,Ed.)\mbox{]}\{\textbackslash{}editor\} 

Should expand to \textquotesingle{}Eds.\textquotesingle{} if the document has more than one editor and \textquotesingle{}Ed.\textquotesingle{} if it only has one.
