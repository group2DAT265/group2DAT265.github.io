\doxysubsection{Detailed Description}
This worker checks if there is a new version of Jab\+Ref available. If there is it will display a dialog to the user offering him multiple options to proceed (see changelog, go to the download page, ignore this version, and remind later).

If the versions check is executed manually and this is the latest version it will also display a dialog to inform the user.
