\doxysubsection{Detailed Description}
A file output stream that is similar to the standard \mbox{\hyperlink{}{File\+Output\+Stream}}, except that all writes are first redirected to a temporary file. When the stream is closed, the temporary file (atomically) replaces the target file.

In detail, the strategy is to\+: 
\begin{DoxyEnumerate}
\item Write to a temporary file (with .tmp suffix) in the same directory as the destination file. 
\item Create a backup (with .bak suffix) of the original file (if it exists) in the same directory. 
\item Move the temporary file to the correct place, overwriting any file that already exists at that location. 
\item Delete the backup file (if configured to do so). 
\end{DoxyEnumerate}If all goes well, no temporary or backup files will remain on disk after closing the stream. 

Errors are handled as follows\+: 
\begin{DoxyEnumerate}
\item If anything goes wrong while writing to the temporary file, the temporary file will be deleted (leaving the original file untouched). 
\item If anything goes wrong while copying the temporary file to the target file, the backup of the original file is kept. 
\end{DoxyEnumerate}

Implementation inspired by code from \href{https://github.com/martylamb/atomicfileoutputstream/blob/master/src/main/java/com/martiansoftware/io/AtomicFileOutputStream.java}{\texttt{ Marty Lamb}} and \href{https://github.com/apache/zookeeper/blob/master/src/java/main/org/apache/zookeeper/common/AtomicFileOutputStream.java}{\texttt{ Apache}}.
