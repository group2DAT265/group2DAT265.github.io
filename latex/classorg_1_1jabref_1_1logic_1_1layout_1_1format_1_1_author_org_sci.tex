\doxysubsection{Detailed Description}
Will return the \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1layout_1_1format_1_1_authors}{Authors}} to match the Org\+Sci format\+:


\begin{DoxyItemize}
\item That is the first author is Last\+First, but all others are First\+Last. 
\item First names are abbreviated 
\item Spaces between abbreviated first names are N\+OT removed. Use \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1layout_1_1format_1_1_no_space_between_abbreviations}{No\+Space\+Between\+Abbreviations}} to achieve this. 
\end{DoxyItemize}
