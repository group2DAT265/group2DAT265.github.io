\doxysubsection{Detailed Description}
This class represents the removal of entries. The constructor needs references to the database, the entries, and the map of open entry editors. T\+O\+DO is this map still being used? The latter to be able to close the entry\textquotesingle{}s editor if it is opened after an undo, and the removal is then undone.
