\doxysubsection{Detailed Description}
This formatter outputs the input value after adding a prefix and a postfix, as long as the input value is non-\/empty. If the input value is empty, an empty string is output (the prefix and postfix are not output in this case).

The formatter requires an argument containing the prefix and postix separated by a comma. To include a the comma character in either, use an escape sequence (\textbackslash{},).
