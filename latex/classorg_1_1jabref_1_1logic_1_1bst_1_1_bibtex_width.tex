\doxysubsection{Detailed Description}
The $\vert$built\+\_\+in$\vert$ function \{.\{purify\$\}\} pops the top (string) literal, removes nonalphanumeric characters except for $\vert$white\+\_\+space$\vert$ and $\vert$sep\+\_\+char$\vert$ characters (these get converted to a $\vert$space$\vert$) and removes certain alphabetic characters contained in the control sequences associated with a special character, and pushes the resulting string. If the literal isn\textquotesingle{}t a string, it complains and pushes the null string.
