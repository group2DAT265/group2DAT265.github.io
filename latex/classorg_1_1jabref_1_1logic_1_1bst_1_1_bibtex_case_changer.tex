We\textquotesingle{}re dealing with a special character (usually either an undotted `\textbackslash{}i' or `\textbackslash{}j', or an accent like one in Table$\sim$3.1 of the \textbackslash{}\+La\+TeX\textbackslash{} manual, or a foreign character like one in Table$\sim$3.2) if the first character after the $\vert$left\+\_\+brace$\vert$ is a $\vert$backslash$\vert$; the special character ends with the matching $\vert$right\+\_\+brace$\vert$. How we handle what is in between depends on the special character. In general, this code will do reasonably well if there is other stuff, too, between braces, but it doesn\textquotesingle{}t try to do anything special with $\vert$colon$\vert$s.
