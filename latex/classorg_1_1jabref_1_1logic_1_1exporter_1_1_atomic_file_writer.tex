\doxysubsection{Detailed Description}
Writer that similar to the built-\/in \mbox{\hyperlink{}{java.\+io.\+File\+Writer}} but uses the \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1exporter_1_1_atomic_file_output_stream}{Atomic\+File\+Output\+Stream}} as the underlying output stream. In this way, we make sure that the errors during the write process do not destroy the contents of the target file. Moreover, this writer checks if the chosen encoding supports all text that is written. Characters whose encoding was problematic can be retrieved by \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1exporter_1_1_atomic_file_writer_a20b8c870d27e09cbd3144c0de79b9d25}{get\+Encoding\+Problems()}}.
