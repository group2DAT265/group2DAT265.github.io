\doxysubsection{Detailed Description}
Facade to unify the access to the citation style engine. Use these methods if you need rendered Bib\+TeX item(s) in a given journal style. This class uses \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1citationstyle_1_1_c_s_l_adapter}{C\+S\+L\+Adapter}} to create output.
