\doxysubsection{Detailed Description}
This class can be used to wrap an \begin{DoxySeeAlso}{See also}
Observable\+List inside it. When wrapped, \mbox{\hyperlink{org_8citationstyles_8styles_8txt_a1bfeace28cb7fe52f211ded8e37acab0}{any}} Listener listening \mbox{\hyperlink{org_8citationstyles_8styles_8txt_ab823061043c5cc565ab7bfa882774562}{for}} updates to \mbox{\hyperlink{microba_8txt_a32d1e5f290ae37ab4d28c5484c6076ad}{the}} wrapped Observable\+List (\mbox{\hyperlink{org_8citationstyles_8styles_8txt_ab823061043c5cc565ab7bfa882774562}{for}} \mbox{\hyperlink{org_8citationstyles_8styles_8txt_a270c8fdeba3943811a4cf9a482e73642}{example}} because \mbox{\hyperlink{org_8citationstyles_8styles_8txt_af796eca1ee2c70f9dd3321aadcf20bfe}{of}} \mbox{\hyperlink{_readme_8txt_a4a72c9f4aeead17626fb9869ba1e4f1f}{a}} binding to it) is ensured to be notified \mbox{\hyperlink{de_8undercouch_8citeproc-java_8txt_a99e0c7c6bd1082fe4cf97f0912e847d7}{on}} \mbox{\hyperlink{microba_8txt_a32d1e5f290ae37ab4d28c5484c6076ad}{the}} Java\+FX Application Thread. It should be used to implement bindings where updates come \mbox{\hyperlink{org_8citationstyles_8styles_8txt_a0297d2e7bc1126f2808896e7c19c7310}{in}} \mbox{\hyperlink{_biblioscape_importer_test_optional_fields_8txt_a958a754f010f8c7c8bc1fc3877cc1743}{from}} \mbox{\hyperlink{_readme_8txt_a4a72c9f4aeead17626fb9869ba1e4f1f}{a}} background thread but should be reflected \mbox{\hyperlink{org_8citationstyles_8styles_8txt_a0297d2e7bc1126f2808896e7c19c7310}{in}} \mbox{\hyperlink{microba_8txt_a32d1e5f290ae37ab4d28c5484c6076ad}{the}} UI where it is necessary \mbox{\hyperlink{_biblioscape_importer_test_keywords_8txt_ab4bf8743b352c7e4728dc399514c3b00}{that}} changes to \mbox{\hyperlink{microba_8txt_a32d1e5f290ae37ab4d28c5484c6076ad}{the}} UI are performed \mbox{\hyperlink{de_8undercouch_8citeproc-java_8txt_a99e0c7c6bd1082fe4cf97f0912e847d7}{on}} \mbox{\hyperlink{microba_8txt_a32d1e5f290ae37ab4d28c5484c6076ad}{the}} Java\+FX Application thread.
