Parses the next token. 

The string being parsed is stored in global variable {\ttfamily original}, and position which parsing has to start from is stored in global variable {\ttfamily token\+\_\+end}; thus, {\ttfamily token\+\_\+end} has to be set to 0 before the first invocation. Procedure updates {\ttfamily token\+\_\+end}; thus, subsequent invocations do not require any additional variable settings. 

The type of the token is returned; if it is {\ttfamily \mbox{\hyperlink{enumorg_1_1jabref_1_1logic_1_1importer_1_1_author_list_parser_1_1_token_ac27c91cee209985d08100bb1c41e13a2}{Token.\+W\+O\+RD}}}, additional information is given in global variables {\ttfamily token\+\_\+start}, {\ttfamily token\+\_\+end}, {\ttfamily token\+\_\+abbr}, {\ttfamily token\+\_\+term}, and {\ttfamily token\+\_\+case}; namely\+: {\ttfamily original.\+substring(token\+\_\+start,token\+\_\+end)} is the text of the token, {\ttfamily original.\+substring(token\+\_\+start,token\+\_\+abbr)} is the token abbreviation, {\ttfamily token\+\_\+term} contains token terminator (space or dash), and {\ttfamily token\+\_\+case} is {\ttfamily true}, if token is upper-\/case and {\ttfamily false} if token is lower-\/case.
