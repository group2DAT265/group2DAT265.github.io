\doxysubsection{Detailed Description}
Provides simple checks to ensure the correct version for Jab\+Ref is available. Currently, we need to make sure that we have Java 1.\+8 but not Java 9. The functions here are not intended for direct use. Instead, they are called inside \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1util_1_1_build_info}{Build\+Info}}, which has the required java version string (e.\+g. 1.\+8.\+0\+\_\+144) available through the build system. The version check should always happen through the {\ttfamily Globals\+::\+B\+U\+I\+L\+D\+\_\+\+I\+N\+FO} instance which is available at a very early stage in the Jab\+Ref startup.
