\doxysubsection{Detailed Description}
Fetch or search from Pub\+Med \href{http://www.ncbi.nlm.nih.gov/sites/entrez/}{\texttt{ www.\+ncbi.\+nlm.\+nih.\+gov}} The \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1importer_1_1fetcher_1_1_medline_fetcher}{Medline\+Fetcher}} fetches the entries from the Pub\+Med database. See \href{https://docs.jabref.org/import-export/medlineris}{\texttt{ docs.\+jabref.\+org}} for a detailed documentation of the available fields.

Performs a cleanup of the fetched entry.

Only systematic errors of the fetcher should be corrected here (i.\+e. if information is consistently contained in the wrong field or the wrong format) but not cosmetic issues which may depend on the user\textquotesingle{}s taste (for example, LateX code vs H\+T\+ML in the abstract).

Try to reuse existing \mbox{\hyperlink{}{Formatter}} for the cleanup. For example, 
\begin{DoxyCode}{0}

\end{DoxyCode}
