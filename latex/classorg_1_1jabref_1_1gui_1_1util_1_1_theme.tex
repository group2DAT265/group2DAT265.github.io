\doxysubsection{Detailed Description}
Installs the style file and provides live reloading. Jab\+Ref provides two inbuilt themes and a user customizable one\+: Light, Dark and Custom. The Light theme is basically the base.\+css theme. Every other theme is loaded as an addition to base.\+css.
