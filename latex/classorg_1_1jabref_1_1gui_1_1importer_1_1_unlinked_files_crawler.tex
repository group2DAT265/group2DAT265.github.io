Searches recursively all files in the specified directory. ~\newline
 ~\newline
 All files matched by the given \mbox{\hyperlink{classorg_1_1jabref_1_1gui_1_1importer_1_1_unlinked_p_d_f_file_filter}{Unlinked\+P\+D\+F\+File\+Filter}} are taken into the resulting tree. ~\newline
 ~\newline
 The result will be a tree structure of nodes of the type \mbox{\hyperlink{}{Check\+Box\+Tree\+Item}}. ~\newline
 ~\newline
 The user objects that are attached to the nodes is the \mbox{\hyperlink{}{File\+Node\+Wrapper}}, which wraps the \mbox{\hyperlink{}{File}}-\/Object. ~\newline
 ~\newline
 For ensuring the capability to cancel the work of this recursive method, the first position in the integer array \textquotesingle{}state\textquotesingle{} must be set to 1, to keep the recursion running. When the states value changes, the method will resolve its recursion and return what it has saved so far.
