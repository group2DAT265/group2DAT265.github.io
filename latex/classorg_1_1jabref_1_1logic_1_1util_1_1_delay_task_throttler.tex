\doxysubsection{Detailed Description}
This class allows to throttle a list of tasks. Use case\+: you have an event that occurs often, and every time you want to invoke the same task. However, if a lot of events happen in a relatively short time span, then only one task should be invoked. 
\begin{DoxyParams}{Parameters}
