We welcome contributions to the Citation Style Language (C\+SL) locale files in this repository.

The C\+SL locale files provide the standard translations for automatic localization of C\+SL styles. As such, the C\+SL locale files should generally contain the most commonly used translations for a given locale. Less common translations can be \href{https://docs.citationstyles.org/en/stable/specification.html\#locale}{\texttt{ provided in the individual C\+SL styles}} that require them, which will overwrite the C\+SL locale file translations.

Because each C\+SL locale file offers the standard translations for all C\+SL styles, changes should be made conservatively and carefully. We will often ask a second native speaker and/or past contributor to look over your proposed changes.\hypertarget{md_src_main_resources_csl-locales__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md59}{}\doxysection{Licensing and crediting}\label{md_src_main_resources_csl-locales__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md59}
