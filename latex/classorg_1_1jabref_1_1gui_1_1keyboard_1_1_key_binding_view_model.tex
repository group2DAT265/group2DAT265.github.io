\doxysubsection{Detailed Description}
This class represents a view model for objects of the \mbox{\hyperlink{enumorg_1_1jabref_1_1gui_1_1keyboard_1_1_key_binding}{Key\+Binding}} class. It has two properties representing the localized name of an action and its key bind. It can also represent a key binding category instead of a key bind itself.
