Performs a cleanup of the fetched entry. 

Only systematic errors of the fetcher should be corrected here (i.\+e. if information is consistently contained in the wrong field or the wrong format) but not cosmetic issues which may depend on the user\textquotesingle{}s taste (for example, LateX code vs H\+T\+ML in the abstract). 

Try to reuse existing \mbox{\hyperlink{}{Formatter}} for the cleanup. For example, 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{new} FieldFormatterCleanup(StandardField.TITLE, \textcolor{keyword}{new} RemoveBracesFormatter()).cleanup(\mbox{\hyperlink{namespacejabref_host_ae5a8819641ec01f8c9f4777dabf9e6a0}{entry}}); }
\end{DoxyCode}
 

By default, no cleanup is done.
