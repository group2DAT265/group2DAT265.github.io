\doxysubsection{Detailed Description}
Saves the given \mbox{\hyperlink{}{Bib\+Database\+Context}} on every \mbox{\hyperlink{}{Bib\+Database\+Context\+Changed\+Event}} by posting a new \mbox{\hyperlink{}{Autosave\+Event}}. An intelligent \mbox{\hyperlink{}{Scheduled\+Thread\+Pool\+Executor}} prevents a high load while saving and rejects all redundant save tasks. The scheduled action is stored and canceled if a newer save action is proposed.
