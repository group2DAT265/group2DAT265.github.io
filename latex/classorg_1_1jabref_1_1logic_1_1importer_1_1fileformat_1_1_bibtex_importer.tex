Parse the database in the source. 

This method can be called in two different contexts -\/ either when importing in a specified format, or when importing in unknown format. In the latter case, Jab\+Ref cycles through all available import formats. No error messages or feedback is displayed from individual import formats in this case. 

If importing in a specified format and an empty library is returned, Jab\+Ref reports that no entries were found. 

This method should never return null.
