\doxysubsection{Detailed Description}
Action for the \char`\"{}\+Save\char`\"{} and \char`\"{}\+Save as\char`\"{} operations called from \mbox{\hyperlink{classorg_1_1jabref_1_1gui_1_1_base_panel}{Base\+Panel}}. This class is also used for save operations when closing a database or quitting the applications. 

The save operation is loaded off of the G\+UI thread using \mbox{\hyperlink{}{Background\+Task}}. Callers can query whether the operation was canceled, or whether it was successful.
