\doxysubsection{Detailed Description}
Inserts the current date (the time a database is being exported).

If a field\+Text is given, it must be a valid \mbox{\hyperlink{}{Date\+Time\+Formatter}} pattern. If none is given, the format pattern will be {\ttfamily yyyy-\/\+M\+M-\/dd hh\+:mm\+:ss z}. This follows I\+S\+O-\/8601. Reason\+: \href{https://xkcd.com/1179/}{\texttt{ https\+://xkcd.\+com/1179/}}.
