\doxysubsection{Detailed Description}
Represents a node in a tree. 

Usually, tree nodes have a value property which allows access to the value stored in the node. In contrast to this approach, the \mbox{\hyperlink{classorg_1_1jabref_1_1model_1_1_tree_node}{Tree\+Node}}$<$T$>$ class is designed to be used as a base class which provides the tree traversing functionality via inheritance. 

Example usage\+: private class Basic\+Tree\+Node extends \mbox{\hyperlink{classorg_1_1jabref_1_1model_1_1_tree_node}{Tree\+Node}}$<$Basic\+Tree\+Node$>$ \{ public Basic\+Tree\+Node() \{ super(Basic\+Tree\+Node.\+class); \} \} 

This class started out as a copy of javax.\+swing.\+tree.\+Default\+Mutable\+Tree\+Node.
