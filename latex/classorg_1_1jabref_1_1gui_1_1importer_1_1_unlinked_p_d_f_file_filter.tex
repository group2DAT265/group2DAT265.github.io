\doxysubsection{Detailed Description}
\mbox{\hyperlink{}{File\+Filter}} implementation, that allows only files which are not linked in any of the \mbox{\hyperlink{}{Bib\+Entry}}s of the specified \mbox{\hyperlink{}{Bib\+Database}}. ~\newline
 ~\newline
 This \mbox{\hyperlink{}{File\+Filter}} sits on top of another \mbox{\hyperlink{}{File\+Filter}} -\/implementation, which it first consults. Only if this major filefilter has accepted a file, this implementation will verify on that file.
