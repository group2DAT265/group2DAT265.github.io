\doxysubsection{Detailed Description}
Formatter that returns the last page from the \char`\"{}pages\char`\"{} field, if set.

For instance, if the pages field is set to \char`\"{}345-\/360\char`\"{} or \char`\"{}345-\/-\/360\char`\"{}, this formatter will return \char`\"{}360\char`\"{}.
