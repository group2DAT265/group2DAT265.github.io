Helper function for retrieving a Bib\+Entry from the Dublin\+Core metadata in a P\+DF file. 

To understand how to get hold of a Dublin\+Core have a look in the test cases for X\+M\+P\+Util. 

The Bib\+Entry is build by mapping individual fields in the dublin core (like creator, title, subject) to fields in a bibtex bib\+Entry. In case special \char`\"{}bibtex/\char`\"{} entries are contained, the normal dublin core fields take precedence. For instance, the dublin core date takes precedence over bibtex/month.
