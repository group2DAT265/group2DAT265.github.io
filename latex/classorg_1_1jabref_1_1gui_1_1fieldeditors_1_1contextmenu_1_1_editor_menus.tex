\doxysubsection{Detailed Description}
Provides context menus for the text fields of the entry editor. Note that we use \mbox{\hyperlink{}{Supplier}} to prevent an early instantiation of the menus. Therefore, they are attached to each text field but instantiation happens on the first right-\/click of the user in that field. The late instantiation is done by \{\mbox{\hyperlink{}{\}. }}
