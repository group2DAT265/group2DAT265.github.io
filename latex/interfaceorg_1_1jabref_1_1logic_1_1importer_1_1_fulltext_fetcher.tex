\doxysubsection{Detailed Description}
This interface is used for classes that try to resolve a full-\/text P\+DF url for a Bib\+Tex entry. Implementing classes should specialize on specific article sites. See e.\+g. \href{http://libguides.mit.edu/apis}{\texttt{ http\+://libguides.\+mit.\+edu/apis}}\}.
