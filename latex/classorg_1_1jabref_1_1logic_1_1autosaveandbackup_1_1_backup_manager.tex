\doxysubsection{Detailed Description}
Backups the given bib database file from \mbox{\hyperlink{}{Bib\+Database\+Context}} on every \mbox{\hyperlink{}{Bib\+Database\+Context\+Changed\+Event}}. An intelligent \mbox{\hyperlink{}{Executor\+Service}} with a \mbox{\hyperlink{}{Blocking\+Queue}} prevents a high load while making backups and rejects all redundant backup tasks. This class does not manage the .bak file which is created when opening a database.
