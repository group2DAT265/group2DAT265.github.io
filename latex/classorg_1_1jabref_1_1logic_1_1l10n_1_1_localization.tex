\doxysubsection{Detailed Description}
Provides handling for messages and menu entries in the preferred language of the user. 

Notes\+: All messages and menu-\/entries in Jab\+Ref are stored in escaped form like \char`\"{}\+This\+\_\+is\+\_\+a\+\_\+message\char`\"{}. This message serves as key inside the \mbox{\hyperlink{namespaceorg_1_1jabref_1_1logic_1_1l10n}{l10n}} properties files that hold the translation for many languages. When a message is accessed, it needs to be unescaped and possible parameters that can appear in a message need to be filled with values. 

This implementation loads the appropriate language by importing all keys/values from the correct bundle and stores them in unescaped form inside a \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1l10n_1_1_localization_1_1_localization_bundle}{Localization\+Bundle}} which provides fast access because it caches the key-\/value pairs. 

The access to this is given by the functions \mbox{\hyperlink{classorg_1_1jabref_1_1logic_1_1l10n_1_1_localization_aed748bba1971c69c99e98411843efd84}{Localization\#lang(\+String, String...)}} and that developers should use whenever they use strings for the e.\+g. G\+UI that need to be translatable.
