\doxysubsection{Detailed Description}
This is an immutable class that keeps information regarding single author. It is just a container for the information, with very simple methods to access it. 

Current usage\+: only methods {\ttfamily get\+Last\+Only}, {\ttfamily get\+First\+Last}, and {\ttfamily get\+Last\+First} are used; all other methods are provided for completeness.
