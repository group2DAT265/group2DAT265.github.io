\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1jabref_1_1gui_1_1openoffice_1_1_bootstrap}{Bootstrap}} offers functionality to obtain a context or simply a service manager. The service manager can create a few basic services, whose implementations are\+: 
\begin{DoxyItemize}
\item com.\+sun.\+star.\+comp.\+loader.\+Java\+Loader 
\item com.\+sun.\+star.\+comp.\+urlresolver.\+Url\+Resolver 
\item com.\+sun.\+star.\+comp.\+bridgefactory.\+Bridge\+Factory 
\item com.\+sun.\+star.\+comp.\+connections.\+Connector 
\item com.\+sun.\+star.\+comp.\+connections.\+Acceptor 
\item com.\+sun.\+star.\+comp.\+servicemanager.\+Service\+Manager 
\end{DoxyItemize}

Other services can be inserted into the service manager by using its X\+Set interface\+: 
\begin{DoxyPre}
    XSet xSet = UnoRuntime.queryInterface( XSet.class, aMultiComponentFactory );
    // insert the service manager
    xSet.insert( aSingleComponentFactory );
\end{DoxyPre}
