\doxysubsection{Detailed Description}
An Action for launching mass field. 

Functionality\+: 
\begin{DoxyItemize}
\item Defaults to selected entries, or all entries if none are selected. 
\item Input field name 
\item Either set field, or clear field. 
\end{DoxyItemize}
