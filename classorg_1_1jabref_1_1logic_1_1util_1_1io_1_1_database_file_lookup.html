◆ lookupDatabase()

boolean org.jabref.logic.util.io.DatabaseFileLookup.lookupDatabase ( File  file)
inline

Returns whether the File file is present in the database as an attached File to an BibEntry.

To do this, the field specified by the key file will be searched for the provided file for every BibEntry in the database.

For the matching, the absolute file paths will be used.

Parameters
fileA File Object.
Returns
true, if the file Object is stored in at least one entry in the database, otherwise false.
