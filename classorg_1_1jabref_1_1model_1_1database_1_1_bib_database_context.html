Detailed Description

Represents everything related to a BIB file.

The entries are stored in BibDatabase, the other data in MetaData and the options relevant for this file in Defaults.

◆ getFileDirectoriesAsPaths()

List<Path> org.jabref.model.database.BibDatabaseContext.getFileDirectoriesAsPaths ( FilePreferences  preferences)
inline

Look up the directories set up for this database. There can be up to three directory definitions for these files: the database's metadata can specify a general directory and/or a user-specific directory, or the preferences can specify one.

The settings are prioritized in the following order, and the first defined setting is used:

  1. user-specific metadata directory
  2. general metadata directory
  3. preferences directory
  4. BIB file directory
